LLVM Top Level README

You are reading the README file at the top level of the LLVM repository (llvm-project). This document will help you find your way around the repository.

The LLVM project is a collection of sub-projects relating to compiler technology. Each sub-project has its own repository module at the top level of this repository.

The main modules of the repository are:

  • java: The Java Front End
  • llvm: The LLVM Compiler Architecture
  • nightly-test-server: Server side scripts for the nightly tests
  • poolalloc: Pooled Allocation
  • The Stacker Language Front End
  • An LLVM Visualiation Suite
  • A Test Suite For LLVM
